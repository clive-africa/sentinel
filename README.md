# Magnus

Magnus is an Excel-based tool designed to streamline Solvency Assessment and Management (SAM) capital calculations for South African short-term insurers. Currently implemented as an Excel workbook with VBA modules, Magnus will be published as an Excel Add-in to provide seamless integration with existing actuarial workflows.

## Purpose

The South African non-life insurance industry needs robust, modern tools for calculating SAM capital requirements. Magnus aims to address this need by providing:

- **Automated Capital Calculations**: Non-life capital requirement calculations including premium and reserve risk, natural catastrophe risk, factor-based and non-proportional capital requirements
- **Multi-dimensional Analysis**: Support for complex organizational structures and reinsurance arrangements
- **Regulatory Compliance**: Built specifically for SAM requirements applicable to South African insurers
- **Efficiency**: Replaces manual calculation workbooks with automated, auditable functions
